AppKita.id adalah aplikasi web berbasis PWA yang dibangun menggunakan Astro, React.js, dan Tailwind CSS di sisi frontend. Backend dikembangkan dengan Laravel serta menggunakan Redis untuk caching dan MySQL sebagai database utama.

Aplikasi ini memiliki 3 jenis pengguna:

  • Buyer: Menjelajahi produk, membeli, dan melacak pesanan
  • Seller: Mengelola produk dan melihat pesanan dari pembeli
  • Admin: Mengatur kategori, mengelola user dan laporan penjualan

Peran: Fullstack Developer

Teknologi: Astro, React.js, Tailwind CSS, Laravel, Redis, MySQL

Link: https://appkita.id
