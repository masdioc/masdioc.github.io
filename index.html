Appkita.id adalah aplikasi web progresif (PWA) untuk e-commerce yang dibangun menggunakan Astro, React.js, dan Tailwind CSS di sisi frontend, dengan backend Laravel yang mengimplementasikan Redis untuk caching dan menggunakan MySQL sebagai database utama.

Teknologi: Astro, React.js, Tailwind, Laravel, Redis, MySQL
